## Description
An AI-powered subscription service chatbot that manages customer subscriptions using compressed plan details and billing history. The chatbot enables users to view plans, subscribe, upgrade, cancel subscriptions, and check billing records through conversational interactions.

## Features
- View subscription plans
- Subscribe / upgrade / downgrade plans
- Check current subscription status
- View billing history
- Cancel or renew subscriptions
- NLP-based chatbot interaction

## Tech Stack
- Frontend: HTML, CSS
- Backend: Python (Flask)
- Database: MySQL
- AI/NLP: Rule-based intent detection
